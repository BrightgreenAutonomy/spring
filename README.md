# Spring Engineering Website

A responsive one-page static website for Spring, an engineering company focused on prefab houses, decks and site-ready structures.

## Files

- `index.html` — page content and structure
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu and sticky header behaviour

## How to use

Open `index.html` in a browser.

To publish, upload these files to any static host such as Netlify, Vercel, Cloudflare Pages, GitHub Pages, or your normal web server.
